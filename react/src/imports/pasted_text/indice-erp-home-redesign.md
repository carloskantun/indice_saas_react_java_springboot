Rediseñar el Home (dashboard principal) del sistema SaaS "Índice ERP".

CONTEXTO DEL PRODUCTO
Índice es un ERP modular para pequeñas empresas en Latinoamérica.
El sistema organiza la empresa en 4 pilares:

- Personas
- Procesos
- Productos
- Finanzas

El Home es el punto de entrada del sistema y funciona como el centro de control del negocio.

El Home actual ya contiene:
- saludo dinámico al usuario
- sección de KPIs
- sección de favoritos
- módulos básicos
- módulos complementarios

NO eliminar estas secciones.
Mejorar su experiencia visual, jerarquía y funcionalidad.

OBJETIVO
Convertir el Home en un dashboard ejecutivo moderno tipo SaaS.

Debe sentirse como:
"el centro de control del negocio".

ESTRUCTURA PROPUESTA

1. HEADER INTELIGENTE

Mostrar:
- saludo dinámico (ej: Buenas tardes, Nahum)
- fecha y hora
- iconos de acción rápida:
  - notificaciones
  - selector de idioma
  - modo aprendiz
  - perfil de usuario

Diseño:
- barra superior limpia
- iconos redondeados
- estilo SaaS moderno

2. SECCIÓN: TUS KPIs

Mostrar métricas clave del negocio:

- ventas semanales
- utilidad neta
- clientes activos
- empleados activos
- tareas pendientes
- gastos del mes

Diseño:

- cards horizontales
- borde redondeado
- sombra suave
- número grande
- indicador de tendencia (verde o rojo)

Agregar botón:

"Configurar KPIs"

3. SECCIÓN: FAVORITOS

Mostrar módulos marcados como favoritos.

Diseño:

- cards grandes
- icono del módulo
- nombre
- estrella activa

Debe funcionar como acceso rápido.

4. SECCIÓN: MÓDULOS PRINCIPALES

Mostrar los módulos principales del sistema:

- Panel inicial
- Recursos Humanos
- Procesos y tareas
- Gastos
- Caja chica
- Punto de venta
- Ventas
- KPIs

Diseño:

- grid de cards
- icono grande
- nombre del módulo
- borde suave
- hover interactivo

Cada módulo debe usar su color semántico:

- Personas → azul
- Procesos → amarillo
- Productos → naranja
- Finanzas → verde

5. SECCIÓN: MÓDULOS COMPLEMENTARIOS

Mostrar módulos adicionales como:

- mantenimiento
- inventarios
- control de minutas
- limpieza
- lavandería
- transportación
- vehículos
- inmuebles

Diseño:

- grid compacto
- cards pequeñas
- icono del módulo

6. MEJORAS ADICIONALES

Agregar debajo del dashboard:

ACTIVIDAD RECIENTE

Ejemplo:

- nueva tarea creada
- gasto registrado
- nuevo cliente agregado
- empleado registrado

Esto ayuda al usuario a entender qué está pasando en su negocio.

DISEÑO VISUAL

El dashboard debe ser:

- limpio
- profesional
- moderno
- estilo SaaS

Usar:

- cards con border-radius
- sombras suaves
- espacios consistentes
- iconografía clara
- diseño responsive

El Home debe transmitir:

"control total del negocio en un solo lugar".
