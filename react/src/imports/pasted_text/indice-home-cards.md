CONTEXTO DEL PROYECTO

Estás diseñando la interfaz del ERP "Índice".

Índice es un sistema modular diseñado para ayudar a las pequeñas y medianas empresas a administrar su negocio de forma clara, organizada y progresiva.

El sistema se basa en una metodología simple de gestión empresarial basada en cuatro pilares:

Personas
Procesos
Productos
Finanzas

El Home del sistema muestra los módulos principales como tarjetas (cards) visuales que permiten acceder rápidamente a cada área del negocio.

Cada módulo tiene un color semántico que comunica su función dentro del sistema.

El diseño debe sentirse:

moderno
limpio
claro
profesional
didáctico

El objetivo es que incluso un empresario sin experiencia tecnológica entienda rápidamente para qué sirve cada módulo.

------------------------------------------------------------

OBJETIVO DE DISEÑO

Diseñar las cards de los "Módulos Básicos" del Home del sistema Índice.

Cada card debe incluir:

• Icono
• Nombre del módulo
• Descripción corta
• Color semántico
• Interacción hover elegante

Las cards deben verse consistentes, amigables y fáciles de escanear visualmente.

------------------------------------------------------------

ESTRUCTURA DE CADA CARD

Cada tarjeta debe contener:

Icono grande en la parte superior  
Nombre del módulo  
Descripción corta (1 línea)  

Ejemplo visual:

[ icono ]

Recursos Humanos
Gestiona tu equipo de trabajo
