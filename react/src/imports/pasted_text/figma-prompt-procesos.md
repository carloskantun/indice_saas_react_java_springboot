Módulo: Procesos y Tareas (Índice ERP)

Prompt:

Diseña el módulo “Procesos y Tareas” para un ERP SaaS llamado Índice.

El diseño debe reutilizar exactamente la estructura visual del módulo de Recursos Humanos, manteniendo:

layout general

cards

tablas

filtros

header

espaciado

tipografía

botones

Sin embargo, el módulo debe usar el color institucional del módulo Procesos y Tareas (amarillo dorado) como color principal para:

botones primarios

tabs activas

indicadores

badges

iconos destacados

El resultado debe sentirse parte del mismo sistema, pero claramente identificado como otro módulo.
